## 📝 Описание

**Media Downloader Pro** — это мощный и удобный инструмент для скачивания видео и историй с популярных платформ. Создан специально для SMM-специалистов и контент-мейкеров, чтобы быстро сохранять референсы, анализировать конкурентов или архивировать свои вебинары и сторис.

Программа имеет современный, интуитивно понятный интерфейс с тёмной темой и не требует знаний командной строки — просто вставьте ссылки и нажмите кнопку!

## 🚀 Быстрый старт

### Вариант 1: Для пользователей Windows (без Python)

1. Скачайте последнюю версию программы из **[Releases](https://github.com/yourname/media-downloader-pro/releases)**
2. Распакуйте архив
3. Запустите **`Media Downloader Pro.exe`**

### Вариант 2: Для разработчиков и пользователей с Python

```bash
# Клонируем репозиторий
git clone https://github.com/yourname/media-downloader-pro.git
cd media-downloader-pro

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем программу
python main.py
```

## 🎬 Первый запуск
### При первом запуске программа автоматически:

1. Проверит наличие FFmpeg в системе
- Если FFmpeg не найден, предложит скачать его

2. Покажет красивый прогресс-бар скачивания и установки

3. После установки запустится основное окно

## 📝 Примеры использования

### Скачать видео с YouTube:

1. Скопируйте ссылку: https://youtu.be/dQw4w9WgXcQ
2. Вставьте в поле ввода (Ctrl+V)
3. Нажмите "➕ Добавить в очередь"
4. Нажмите "⬇️ Скачать всё"
5. Наблюдайте за прогрессом

### Скачать stories из Instagram:

1. Введите логин и пароль Instagram в блоке авторизации
2. Вставьте ссылку на профиль: https://instagram.com/username
3. Добавьте в очередь и запустите загрузку
4. Программа скачает все активные stories

### Скачать несколько видео сразу:

1. Вставьте ссылки (каждая с новой строки):
-   https://youtu.be/xxxxx
-   https://vk.com/video-xxxxx
-   https://t.me/channel/123
   
2. Нажмите "➕ Добавить в очередь" — все ссылки попадут в список
3. Нажмите "⬇️ Скачать всё" — программа загрузит их по очереди
4. Следите за прогрессом в очереди и логе

## ⚙️ Конфигурация

### Настройка Telegram API
Для скачивания из Telegram по ссылкам на сообщения (например, t.me/.../123) потребуются api_id и api_hash.

1. Получите их на "my.telegram.org"
2. Откройте файл config.py
3. Найдите и заполните настройки:

```bash
TELEGRAM_CONFIG = {
    "api_id": 12345,        # Ваш API ID
    "api_hash": "abc...",   # Ваш API Hash
    "session_name": "telegram_session",
}
```
* 💡 Примечание: Публичные ссылки на каналы работают без авторизации.

### Изменение темы

* В файле config.py можно изменить тему оформления:

```bash
python
THEME = {
    "appearance_mode": "dark",     # dark / light / system
    "color_theme": "green",        # blue / green / dark-blue
}
```

### Настройка FFmpeg

* FFmpeg устанавливается автоматически при первом запуске. Если нужно переустановить:

1. Нажмите кнопку "🎬 FFmpeg" в интерфейсе
2. Выберите "Скачать FFmpeg"
3. Дождитесь завершения установки

## 🔧 Устранение неполадок
### ❌ Программа не запускается

| Проблема            | 	Решение|
|---------------------|----------|
| **Python не установлен**     |Скачайте и установите Python 3.9+ с официального сайта
| **Зависимости не установлены**  |Выполните pip install -r requirements.txt |
| **Ошибка прав доступа**        |Запустите от имени администратора |

### 🎬 Проблемы с загрузкой видео

| Проблема                               | 	Решение|
|----------------------------------------|----------|
| **Не скачивается видео (Youtube)**     |Обновите yt-dlp: pip install --upgrade yt-dlp
| **Медленная загрузка(Youtube)**        |Выберите меньшее качество в настройках |
| **Не скачиваются stories (Instagram)** |Проверьте логин/пароль, войдите в браузер |
| **Не скачиваются файлы(Telegram)**     |  Проверьте API_ID и API_HASH       |

### 🔧 Ошибки FFmpeg

| Ошибка                               | 	Решение                                            |
|---------------------------------------|-----------------------------------------------------|
| **FFmpeg не найден**    | Нажмите кнопку "🎬 FFmpeg" и скачайте автоматически 
| **Ошибка при конвертации**       | Переустановите FFmpeg через кнопку в интерфейсе     |
| **Не скачивается FFmpeg** | Проверьте подключение к интернету/используйте vpn   |

## ❓ Часто задаваемые вопросы
### Почему не скачивается видео с Telegram?

* Убедитесь, что:

1. Ссылка ведёт на публичное сообщение с медиа (например, https://t.me/durov/123)
2. Для приватных каналов настроены API ключи (см. раздел Конфигурация)

### Не работает Instagram Stories

* Возможные причины:

1. Неправильный логин/пароль
2. Instagram требует подтверждения — войдите через браузер
3. Слишком частые запросы — сделайте паузу

### Хочу скачать только аудио

* В настройках формата выберите audio. Программа сохранит звуковую дорожку в формате MP3.

### Программа зависает при загрузке

* Обычно это связано с сетью:

1. Подождите немного — yt-dlp пытается переподключиться
2. Проверьте скорость интернета
3. Перезапустите программу

### Как обновить программу?

```bash
git pull
pip install --upgrade -r requirements.txt

```

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --icon=icon.ico main.py
```
